LESS - программа-читалка по умолчанию.

Команды:
* `/ <выражение>` - поиск по выражению.
* `n` - поиск вперёд.
* `shift + n` - поиск назад.
* `q` - выход из читалки.

## Полный список команд

* `cat <файл>` - посмотреть файл.
* `ls -a` - показать все файлы и скрытые.
* `tree -a` - отобразить дерево директории.
* `rm <файл>` - удалить файл.
